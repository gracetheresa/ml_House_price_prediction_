 Project Description--

This project is a Streamlit-based web application that predicts house prices using Simple Linear Regression.
The model is trained on generated sample data where house price depends on house size.
Users can enter a house size and instantly get a predicted price along with a visual representation.

Features--

Interactive web interface using Streamlit

Uses Linear Regression for prediction

Generates sample house data automatically

Visualizes data using Plotly

Highlights predicted price on the graph

Technologies Used--

Python

Streamlit

NumPy

Pandas

Scikit-learn

Plotly
